# Web Application

Next.js frontend for the Enterprise Cold Email & Outreach Platform.

## Responsibilities

* Authentication (Login, Register, Profile Management)
* Dashboard & Analytics
* Lead Management
* CSV Lead Import
* Lead History Timeline
* AI Email Draft Generation
* Campaign Management
* Email Sending Interface
* SMTP Configuration
* Daily Limit & Ramp-Up Settings
* Email Activity Tracking
* Open & Click Analytics
* Delivery Status Monitoring
* Queue Monitoring
* User Settings

## Technology

* Next.js
* React
* TypeScript
* Tailwind CSS

## Run

```bash
bun run dev
```
